# Student Record Management System

A menu-driven Python application used to manage student records efficiently.

## Features

- Add Student Record
- View Student Records
- Search Student by Roll Number
- Update Student Details
- Delete Student Records
- User-Friendly Menu Interface

## Technologies Used

- Python

## Concepts Used

- Functions
- Lists
- Dictionaries
- Loops
- Conditional Statements
- CRUD Operations

## How to Run

1. Clone the repository
2. Open the project folder
3. Run the following command:

```bash
python main.py
```

## Project Objective

The objective of this project is to demonstrate the implementation of CRUD operations using Python and strengthen problem-solving and logical programming skills.
